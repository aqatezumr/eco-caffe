# Eco Caffe — Курсовой проект

Сайт эко-кафе "Зелёный лист", созданный на WordPress в рамках курсовой работы.

## Содержимое
- Главная страница (тема Cafe Elementor)
- Бронирование (плагин Bookly)
- Меню
- О нас (Форма обратной связи (Kali Forms))
- Адаптивный дизайн

## Установка
1. Скопируйте файлы в папку `htdocs` (если используете XAMPP)
2. Импортируйте базу данных через phpMyAdmin
3. Обновите `wp-config.php` с вашими данными
